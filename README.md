# 🧙 Hearthstone - UNIVALI Saga

Projeto em C++ baseado em combate estilo Hearthstone.

## 💡 Diferencial
- Sem uso de if/else
- Sem uso de loops
- Toda lógica baseada em matemática (0 e 1)

## 🎮 Como funciona
- Escolha cartas
- Buffs e debuffs são aplicados matematicamente
- Vitória depende do cálculo final

## 🚀 Tecnologias
- C++
- Lógica matemática aplicada
